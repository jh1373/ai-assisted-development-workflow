# Review Checklist

AI支援開発で、完了前に確認するためのチェックリストです。

## 実装前レビュー

- 目的は明確か
- 完了条件は明確か
- 変更する範囲は明確か
- 変更しない範囲は明確か
- 既存仕様との矛盾はないか
- テスト方法は決まっているか
- セキュリティやプライバシーへの影響はないか

## 実装中レビュー

- AIの提案をそのまま採用していないか
- 既存コードのパターンに沿っているか
- 不要なリファクタリングが混ざっていないか
- 1つの仮説ごとに検証しているか
- エラー原因を推測だけで処理していないか

## 完了前レビュー

- 計画したタスクは完了しているか
- 完了条件を満たしているか
- テストやビルドは通っているか
- 手動確認が必要な箇所を確認したか
- 余計なファイルを変更していないか
- ドキュメント更新が必要な箇所を更新したか
- devlogに判断理由を残したか
- 次回作業者が再開できる状態か

## 公開前レビュー

- APIキー、トークン、パスワードが含まれていないか
- 個人情報が含まれていないか
- 管理画面URLや社内固有情報が含まれていないか
- スクリーンショットやログに秘密情報が写っていないか
- READMEだけで目的が伝わるか
- 初見の人が使い方を理解できるか
