# プロジェクト案内図 可読性改善 計画

## 目的

小さな文字と高い情報密度を解消し、初心者が拡大操作なしで内容と操作方法を理解できる画面にする。

## 対象

- 全画面の文字サイズと行間
- 補足文字と背景のコントラスト
- メニュー、ボタン、入力欄、ツリー行の操作領域
- スマートフォンの数値表示
- 画面幅が狭い場合の折り返し

## 完了条件

- 表示文字が12px未満にならない
- 標準本文が16pxになる
- 主要操作領域が44px以上になる
- スマートフォンで数値が1列表示される
- PCとスマートフォンの画像確認、全自動テスト、rootとstarterの同期確認が成功する
