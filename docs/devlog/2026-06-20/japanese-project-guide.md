# プロジェクト案内図の日本語改善

## 変更理由

機能名を英語のまま表示すると、初心者は名前を覚えてから用途を推測する必要がある。画面を見ただけで「何を確認できるか」が分かる表現へ統一した。

## 主な判断

- 製品名のような呼び方より、機能を直接表す`プロジェクト案内図`を採用した
- 割合だけでは実態が分かりにくいため、上部の集計は件数を表示した
- 内部のJSON項目名は互換性のため変更せず、人間が読む表示だけを日本語化した
- 既存導入先を壊さないため、初回設定判定は旧英語表記も受け付ける
