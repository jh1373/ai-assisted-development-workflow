# プロジェクト案内図 UI再設計

## 変更理由

以前の画面は、数字の良し悪しを判断できず、ディレクトリ構成も一般的なファイルツリーと大きく変わらなかった。また、暗色、強いアクセント色、大量のカードにより、ベンチマーク由来の印象と生成AI製UIの定型感が強かった。

## 主な判断

- 暗色テーマと装飾的な見出しを廃止し、長時間読める明るい画面へ変更した
- 数字を単独の評価指標として扱わず、分母と自然文の判定を正本にした
- ディレクトリは、最上位の構成、全ツリー、選択項目の説明という3段階に分けた
- モバイルではメニューを省略せず、5項目すべてを表示した
- ベンチマークからは「意味から探索する」思想だけを採用し、色、レイアウト、表現は独自に設計した

## 検証

自動テストに加え、localhost画面を複数の画面幅で撮影して、情報の欠落と折り返しを確認した。
